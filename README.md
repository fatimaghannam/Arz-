🍽️ ARZE – Premium Lebanese Restaurant Website

A modern, responsive restaurant website designed to provide an elegant digital dining experience. ARZE showcases a premium Lebanese restaurant through an interactive online menu with high-quality food photography, intuitive navigation, and a clean user interface optimized for all devices.

This project demonstrates modern frontend development practices and serves as a portfolio project that can be customized for real restaurants.


✨ Features

🍴 Interactive digital restaurant menu
📱 Fully responsive design for desktop, tablet, and mobile
🖼️ High-quality food photography for every menu item
📂 Well-organized menu categories
🛒 Shopping cart with real-time total price calculation
⏰ Restaurant opening hours
📍 Restaurant location section
📞 Contact information
⚡ Fast and responsive user interface
🎨 Modern Lebanese-inspired design
📷 QR-code ready for restaurant table menus


🛠️ Built With

* React
* Vite
* JavaScript (ES6+)
* Material UI (MUI)
* Radix UI
* CSS3
* HTML5



This project is intended for educational, portfolio, and demonstration purposes.
