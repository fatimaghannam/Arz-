Arzé Restaurant Website

Arzé is a modern, responsive restaurant website designed for Lebanese restaurants to showcase their menu in a clean and elegant way. The project provides customers with an easy-to-use digital menu that can be accessed through a QR code, allowing them to browse dishes, view prices, and calculate their order total before ordering inside the restaurant or by phone.

✨ Features
🍽️ Modern and responsive restaurant landing page
📖 Bilingual support (English & Arabic)
📱 QR Code digital menu
🔍 Live search for dishes
🗂️ Category-based menu filtering
🛒 Cart for total price calculation
💰 Real-time total price updates
📍 Restaurant location and opening hours
📞 Call-to-order integration
🖼️ High-quality food photography
📱 Mobile-friendly design
⚡ Fast and lightweight interface

🛠️ Technologies Used
-Frontend: React 19, TypeScript
-Build Tool: Vite
-Styling: Tailwind CSS
-UI Components: Radix UI
-Icons: Lucide React, Material UI Icons
-Routing: React Router
-Forms: React Hook Form
-Animations: Motion
-Charts: Recharts
-Carousel: Embla Carousel
-Design: Responsive UI/UX

🎯 Purpose
This project was created as a realistic restaurant solution that helps businesses replace printed menus with an attractive digital experience. Customers can browse the full menu, calculate the total cost of their meal and order their favorite dishes either in restaurant or through mobile number.
